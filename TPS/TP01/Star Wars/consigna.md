## Caso elegido 3: App Star Wars para estudiantes.

<br>

### Prompt original
"Haceme una página de Star Wars."
<br>

### Prompt mejorado
"Haceme una página de Star Wars con HTML/CSS/JS o react que actue como una wiki interactiva, API Publica utilizando esta para la informacion y ampliar con informacion de la wiki de star wars, que explique a detalle con imagenes (personajes,planetas,naves,especies,organizaciones,peliculas,series, star wars legends y el cannon),que tenga diseño inspirado en la ciencia ficcion, con pantalla de carga y en caso de error que muestre el error, manejo de carga y error."
<br>

### Qué información agregué
Agregué varios elementos al prompt original siendo:
